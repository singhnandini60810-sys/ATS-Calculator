# ATS Resume Analyzer

A simple browser-based ATS (Applicant Tracking System) Resume Analyzer that checks how well a resume matches a job description.

The project works completely on the frontend and can be hosted directly using GitHub Pages.

---

## Features

- Upload resume in PDF format
- Extract text from PDF in browser
- Compare resume with job description
- Calculate ATS compatibility score
- Detect matching skills
- Show missing skills
- Provide improvement suggestions
- No backend required

---

## Technologies Used

- HTML5
- CSS3
- JavaScript
- PDF.js

## How It Works

1. Upload your resume PDF
2. Paste the job description
3. Click Analyze Resume
4. The application:
   - Reads resume text
   - Checks important keywords
   - Calculates ATS score
   - Shows missing skills
